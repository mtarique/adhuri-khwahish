# Adhuri Khwahish Website

This is a simple single-page website built with HTML, CSS, and Bootstrap.

## Setup Instructions

1.  **Images**:
    *   Place your background image in `assets/images/` and rename it to `background.jpg` (or update `style.css` line 7 with the correct filename).
    *   Place your logo image in `assets/images/` and rename it to `logo.png` (or update `index.html` line 21 with the correct filename).

2.  **Links**:
    *   Update the `href="#"` in `index.html` (lines 37 and 38) with your actual Facebook and Instagram profile URLs.

## Hosting on GitHub Pages

1.  Push this code to your GitHub repository.
2.  Go to the repository **Settings**.
3.  Click on **Pages** in the left sidebar.
4.  Under **Source**, select `Deploy from a branch`.
5.  Under **Branch**, select `main` (or `master`) and `/ (root)`.
6.  Click **Save**.
7.  Wait a minute or two, and GitHub will provide you with the live URL of your website.
